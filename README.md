## インストール
Chapter2の全パッケージを以下のコマンドでインストールします．
- ROSのワークスペースを`~/airobot_ws`とする．
  ```
  cd ~/airobot_ws/src
  ```

- Chapter2のリポジトリを入手
  ```
  git clone https://github.com/AI-Robot-Book/chapter2
  ```
  
- パッケージのビルド   
  ```
  cd ~/airobot_ws  
  colcon build
  ```
