# happy_pub_sub　
## 概要
２章のパブリッシュとサブスクライバがあるサンプルプログラム


## インストール
Chapter2のパッケージは全部まとめてインストール・ビルドをします．
- [第2章 インストール](https://github.com/AI-Robot-Book/chapter2)を参照してください．


## 実行  
端末で以下のコマンドを実行
```
cd ~/airobot_ws
source install/setup.bash
ros2 run happy_pub_sub happy_pub_sub_node
```
